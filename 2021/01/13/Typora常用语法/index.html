Typora常用语法

发表于2021-01-13|更新于2022-01-12
|阅读量:

Markdown 常用结构体

1.1 标题

使用 # 或者 ctrl+数字 例如：ctrl + 1 对应一级标题

1.2 段落

ctrl + 0

1.3 区块引用

在段落的每行或者只在第一行使用符号>,还可使用多个嵌套引用，如：

区块应用 >

>嵌套引用

1.4 代码区块

ctrl + shift + K

1.5 强调

在强调内容两侧分别加上*或者_，如：

斜体 ，斜体 ctrl + I

粗体 ctrl + B

1.6 列表

有序使用：ctrl+shift+[

  1. 第一项
  2. 第二项
  3. 第三项

无序使用：ctrl+shift+]

  • 无序列表第一
  • 第二

1.7 分割线

分割线最常使用就是三个或以上*，还可以使用-和_。

1.8 链接

okeyia的github

1.9 图片

ctrl + shift +i

1.10 反斜杠

\相当于反转义作用，使符号成为普通符号.

1.11 符号 `

ctrl + shift + `

起到标记作用，用于字母或数字，例如：abc，123 。
