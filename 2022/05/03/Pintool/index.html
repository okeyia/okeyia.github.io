一 Inter pin 使用简介

其实在研一上学期，就接触pin了，但仅仅只停留在理论上，没有实践。本次主要使用pin来获取某一个段代码访问内存的trace。官网给出的examples中有一个pinatrace.cpp, 只能获取到整个代码的访问trace, 因此需要对pintrace.cpp做出一个更改。

现在pin版本已经更新到了pin 3.22, 下载链接: Pin - A Binary Instrumentation Tool - Downloads

由于网上可以搜到的大多pintools工具都是几年前的, 因此提供一个旧版本的pin下载链接: pin 3.5 下载链接 pin 3.7 下载

用户手册: Pin 3.22 User Guide
