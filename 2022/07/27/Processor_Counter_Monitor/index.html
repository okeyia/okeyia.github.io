Processor-Counter-Monitor测量cache及带宽

Performance Counter Monitor（PCM）是一个由英特尔开发的，也是基于PMU(performance monitoring unit)一个性能检测工具。

它是运行在msr 内核模块(perf是内核系统调用吧？)上的，提供了C++ API。

  • pcm：CPU、cache
  • pcm-memory：内存带宽（每频道及每个DRAM DIMM槽）
  • pcm-latency：L1 cahce
  • pcm-pcie：PCIe
  • pcm-numa：NUMA架构
  • pcm-power：能耗

最新版 , 复现论文[Saga-Bench: Software and Hardware Characterization of Streaming Graph Analytics Workloads] 的过程中, 用到的历史版本 。

PCM Error: can’t open MSR handle for core 0

Try to execute ‘modprobe msr’ as root user and then
you also must have read and write permissions for /dev/cpu//msr devices (/dev/msr for Android). The ‘chown’ command can help.
